# Dotfiles



## Requirements
+ `wget`
+ `curl`
+ `zsh` combined with [oh-my-zsh](https://github.com/robbyrussell/oh-my-zsh)
+ `vim`
+ `tmux`

## Installation

1. `wget https://raw.githubusercontent.com/simplay/.cfg/master/.cfg_install/install.sh`
2. Run the installation shell script: `./install.sh`
